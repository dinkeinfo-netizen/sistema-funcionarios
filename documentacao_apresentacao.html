🎯 Visão Geral

O Sistema de Controle de Acesso de Funcionários é uma solução completa e robusta desenvolvida para gerenciar o controle de acesso e ponto eletrônico de colaboradores em empresas de todos os portes.

Com múltiplas formas de autenticação, interface intuitiva e relatórios detalhados, o sistema oferece flexibilidade, segurança e eficiência para diferentes cenários empresariais.

🔐 Métodos de Autenticação

Reconhecimento Facial

Tecnologia: OpenCV + face_recognition

  • Processamento em tempo real
  • Normalização de iluminação (CLAHE)
  • Confiança mínima configurável (60%)
  • Cache de encodings para performance
  • Proteção contra spam (rate limiting)

Acesso RFID

Suporte: Cartões, tags e chaveiros

  • Leitura instantânea
  • Cadastro via interface administrativa
  • Validação automática
  • Verificação de status ativo
  • Feedback visual e sonoro

Acesso Manual

Método: Número de registro

  • Interface simples e intuitiva
  • Validação em tempo real
  • Verificação de status do funcionário
  • Seleção de tipo de acesso
  • Ideal para backup

QR Code

Geração: QR codes únicos por funcionário

  • Geração automática com dados do funcionário
  • Leitura via câmera do dispositivo
  • Validação instantânea
  • Exibição de nome e departamento
  • Download e impressão disponíveis
Controle Administrativo: O administrador pode habilitar ou desabilitar cada método de acesso individualmente através do painel de configurações.

📋 Informações Técnicas

Requisitos do Sistema

  • SO: Linux (Ubuntu 20.04+ recomendado)
  • Docker: Versão 20.10+
  • Docker Compose: Versão 2.0+
  • RAM: Mínimo 4GB (8GB recomendado)
  • Armazenamento: Mínimo 20GB livre
  • Câmera: Para funcionalidade facial

Instalação Rápida

# Clone do repositório git clone <url-do-repositorio> cd sistema-funcionarios # Iniciar com Docker docker-compose up -d --build # Acessar sistema # http://localhost:8081
